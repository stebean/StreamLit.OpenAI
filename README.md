# StreamLit.OpenAI

StreamLit chatbot y ubicación.

## Instalación

En la terminal:

```bash
$ git clone https://github.com/estebansg1/StreamLit.OpenAI.git
```

Librerias usadas en requirements.txt:
```bash
$ pip install -r requirements.txt"
```
